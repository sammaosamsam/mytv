# 爱家影视 (原Selene) 修复说明

## 已完成的修复

### 1. 切换直播频道闪退BUG修复

#### 问题原因
在切换直播频道时，由于播放器资源释放和初始化的竞态条件，导致应用在快速切换频道时100%闪退。

#### 修复内容

**1. video_player_widget.dart**
- 改进了 `didUpdateWidget` 方法，添加状态检查并使用微任务延迟更新，避免快速切换时的竞态条件
- 增强了 `_updateDataSource` 方法，添加多重状态检查：
  - 检查播放器是否已 disposed
  - 检查URL是否真正改变
  - 在打开新媒体前先暂停当前播放
  - 添加100ms延迟确保资源释放
  - 增强错误处理
- 改进了 `_disposePlayer` 方法：
  - 先取消所有订阅再释放资源
  - 先暂停播放器再 dispose
  - 添加错误处理和延迟

**2. live_player_screen.dart**
- 在 `_switchChannel` 方法中添加重复切换检查
- 使用延迟确保UI更新后再加载EPG数据

**3. mobile_player_controls.dart**
- 改进了 `dispose` 方法，添加错误处理
- 增强了 `_listenPlayerStreams` 方法，添加错误处理和空检查

### 2. APP名称修改

已将应用名称从 "Selene" 修改为 "爱家影视"：
- `pubspec.yaml`: 应用包名和描述
- `lib/main.dart`: 应用标题和Windows窗口标题
- `android/app/src/main/AndroidManifest.xml`: Android应用标签

## 如何构建Android APK

### 环境要求
1. 安装 Flutter SDK (>=3.4.3)
2. 安装 Android SDK
3. 配置好 Flutter 和 Android 环境变量

### 构建步骤

```bash
# 1. 进入项目目录
cd Selene-Source

# 2. 获取依赖
flutter pub get

# 3. 构建Android APK (release版本)
flutter build apk --release \
    --target-platform android-arm64,android-arm \
    --split-per-abi

# 或使用项目提供的构建脚本
./build.sh --android-only
```

### 构建产物
构建完成后，APK文件将位于：
- `build/app/outputs/flutter-apk/app-arm64-v8a-release.apk` (64位)
- `build/app/outputs/flutter-apk/app-armeabi-v7a-release.apk` (32位)

## 修复验证

修复后的代码解决了以下问题：
1. ✅ 播放器资源释放时的竞态条件
2. ✅ 快速切换频道时的重复初始化问题
3. ✅ 流订阅未正确取消导致的内存泄漏

## 注意事项

1. 建议在真机上测试，模拟器可能无法完全模拟视频播放行为
2. 如果仍有闪退问题，请检查直播源URL是否有效
3. 建议在Android 8.0 (API 26) 及以上版本使用
