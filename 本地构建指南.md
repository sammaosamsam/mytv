# 爱家影视 Android APK 本地构建指南

## 环境要求

- macOS 系统（支持 Apple Silicon 和 Intel）
- 约 5GB 可用磁盘空间
- 稳定的网络连接

## 步骤1：安装 Homebrew（如果未安装）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 步骤2：安装 Java JDK

```bash
# 安装 OpenJDK 17（推荐版本）
brew install openjdk@17

# 添加到环境变量
echo 'export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# 验证安装
java -version
```

## 步骤3：安装 Flutter SDK

### 方式A：通过 Homebrew 安装（推荐，最简单）

```bash
# 安装 Flutter
brew install flutter

# 验证安装
flutter doctor
```

### 方式B：手动下载安装

```bash
# 创建开发目录
mkdir -p ~/development
cd ~/development

# 下载 Flutter SDK（Apple Silicon Mac）
curl -o flutter_macos_arm64_3.24.5-stable.zip \
  https://storage.googleapis.com/flutter_infra_release/releases/stable/macos/flutter_macos_arm64_3.24.5-stable.zip

# 解压
unzip flutter_macos_arm64_3.24.5-stable.zip

# 添加到环境变量
echo 'export PATH="$HOME/development/flutter/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# 验证安装
flutter doctor
```

## 步骤4：安装 Android Studio 和 Android SDK

### 安装 Android Studio

```bash
# 通过 Homebrew 安装
brew install --cask android-studio
```

或者手动下载：https://developer.android.com/studio

### 配置 Android SDK

1. 打开 Android Studio
2. 首次启动时选择 "Standard" 安装
3. 等待安装完成

### 安装 Android SDK 组件

```bash
# 打开 Android Studio 的 SDK Manager
# 或通过命令行安装必要组件

# 设置 ANDROID_HOME 环境变量
echo 'export ANDROID_HOME="$HOME/Library/Android/sdk"' >> ~/.zshrc
echo 'export PATH="$ANDROID_HOME/cmdline-tools/latest/bin:$PATH"' >> ~/.zshrc
echo 'export PATH="$ANDROID_HOME/platform-tools:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## 步骤5：配置 Flutter

```bash
# 运行 Flutter doctor 检查环境
flutter doctor

# 根据提示安装缺失的组件
# 通常需要运行：
flutter doctor --android-licenses
```

## 步骤6：构建 APK

```bash
# 进入项目目录
cd /Users/ace/WorkBuddy/20260312162941/Selene-Source

# 获取依赖
flutter pub get

# 构建 Android APK (Release版本，支持arm64和armv7)
flutter build apk --release \
    --target-platform android-arm64,android-arm \
    --split-per-abi

# 或使用项目提供的构建脚本
./build.sh --android-only
```

## 构建产物位置

构建成功后，APK文件位于：

```
build/app/outputs/flutter-apk/
├── app-arm64-v8a-release.apk    (64位设备，推荐)
├── app-armeabi-v7a-release.apk  (32位设备)
└── app-release.apk              (通用版本，较大)
```

## 常见问题

### 1. Flutter doctor 提示 Android SDK 未找到

```bash
# 手动指定 Android SDK 路径
flutter config --android-sdk $HOME/Library/Android/sdk
```

### 2. 构建时提示缺少 NDK

```bash
# 在 Android Studio 中打开 SDK Manager
# 安装 NDK (Side by side) 和 CMake
```

### 3. 构建失败提示内存不足

```bash
# 增加 Gradle 内存限制
# 编辑 android/gradle.properties，添加：
org.gradle.jvmargs=-Xmx4096m
```

### 4. 证书许可问题

```bash
# 接受所有 Android SDK 许可
flutter doctor --android-licenses
# 输入 'y' 接受所有许可
```

## 快速安装脚本（一键安装）

如果你希望快速安装所有依赖，可以使用以下脚本：

```bash
#!/bin/bash

# 安装 Homebrew
if ! command -v brew &> /dev/null; then
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
fi

# 安装 Java
brew install openjdk@17

# 安装 Flutter
brew install flutter

# 安装 Android Studio
brew install --cask android-studio

# 配置环境变量
echo 'export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
echo 'export ANDROID_HOME="$HOME/Library/Android/sdk"' >> ~/.zshrc
echo 'export PATH="$ANDROID_HOME/cmdline-tools/latest/bin:$PATH"' >> ~/.zshrc
echo 'export PATH="$ANDROID_HOME/platform-tools:$PATH"' >> ~/.zshrc
source ~/.zshrc

# 验证安装
flutter doctor

echo "环境安装完成！请重启终端后运行 'flutter doctor' 检查状态"
```

## 构建后的测试

```bash
# 连接安卓设备或启动模拟器
# 安装 APK 到设备
flutter install

# 或手动安装
adb install build/app/outputs/flutter-apk/app-arm64-v8a-release.apk
```

## 发布准备

如果要发布到应用商店，还需要：

1. **签名密钥**：
```bash
# 生成签名密钥
keytool -genkey -v -keystore ~/upload-keystore.jks -keyalg RSA \
    -keysize 2048 -validity 10000 -alias upload
```

2. **配置签名**：在 `android/app/build.gradle` 中配置签名信息

3. **构建发布版本**：
```bash
flutter build apk --release
```
