# 查看 GitHub Actions 构建错误步骤

## 方法1：在浏览器中查看（最简单）

1. 打开 https://github.com/sammaosamsam/mytv/actions
2. 点击最新的红色 ❌ 构建记录
3. 点击 "Build APK" 步骤
4. 查看红色错误信息

## 方法2：复制错误日志给我

在构建日志页面：
1. 找到红色 ❌ 标记的步骤
2. 点击该步骤展开
3. 复制错误信息（通常是红色的文字）
4. 粘贴给我

## 常见错误及解决方案

### 错误1：Java版本问题
```
Error: Could not find or load main class...
```
**解决：** 已修复，使用Java 17

### 错误2：Flutter依赖问题
```
Because selene depends on ...
```
**解决：** 需要检查pubspec.yaml

### 错误3：Android SDK问题
```
Android SDK not found
```
**解决：** 需要配置ANDROID_HOME

### 错误4：构建失败
```
Gradle build failed
```
**解决：** 需要查看详细错误
